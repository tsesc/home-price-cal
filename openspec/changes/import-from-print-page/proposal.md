## Why

使用者目前需要手動逐一輸入實價登錄資料到房價計算器表單中，這個過程繁瑣且容易出錯。內政部實價登錄網站提供列印頁面，包含完整的交易明細（建物面積、車位、價格等）。如果能讓使用者直接從列印頁面複製純文字內容，自動解析並填入表單，可以大幅提升使用體驗。

## What Changes

- 新增「從實價登錄匯入」功能按鈕，開啟文字貼上對話框
- 使用者可將實價登錄列印頁面的全選文字直接貼入
- 自動解析純文字內容，提取：交易總價、車位價格、主建物面積、陽台面積、雨遮面積、共同使用部分面積（區分含車位/不含車位公設）、車位面積、樓別/樓高、土地面積
- 解析結果預覽確認後，自動填入計算器表單
- 自動計算單價（不含車位）= (交易總價 - 車位價格) / 建物總面積（不含車位）

## Capabilities

### New Capabilities
- `print-page-parser`: 解析實價登錄列印頁面純文字內容，提取建物、車位、價格等結構化資料
- `import-dialog`: 匯入對話框 UI 元件，包含文字貼上區域、解析預覽、確認填入功能
